# 大富翁（Monopoly）3D 跨端版

## 项目概述

这是一个基于 Unity 引擎开发的 3D 跨端大富翁游戏，支持 Mac/Windows 平台，实现本地单机多人游戏玩法。

## 核心特性

- 🎮 **本地多人游戏**：支持 2-4 名玩家同屏轮流操作
- 🌐 **跨端兼容**：同一套代码可编译为 Windows (.exe) 和 macOS (.app) 客户端
- 🎲 **经典玩法**：掷骰子、买地、收费、特殊场所、道具卡等核心机制
- 🎨 **3D 视觉**：立体棋盘、动画效果、光影氛围
- 🚀 **模块化设计**：核心逻辑与 UI/3D 表现分离，便于后续扩展

## 技术栈

- **引擎**：Unity
- **语言**：C#
- **目标平台**：Windows 10+、macOS 10.15+（Intel/Apple Silicon）

## 开发阶段

1. **第一阶段**：核心逻辑 + 跨端基础
2. **第二阶段**：3D 视觉完善 + 特殊场所/道具卡
3. **第三阶段**：本地多人完整体验 + 优化
4. **第四阶段**：后期扩展（线上联机、AI 对手、存档等）

## 项目结构

```
Monopoly/
├── Monopoly_Unity_Project/    # Unity 工程根目录
│   ├── Assets/               # 核心资源
│   ├── Packages/             # Unity 包管理
│   ├── ProjectSettings/      # 项目配置
│   └── Build/               # 打包输出目录
├── Docs/                    # 文档目录
└── Scripts/                 # C# 核心脚本
```

## 快速开始

1. 使用 Unity Hub 打开 `Monopoly_Unity_Project` 目录
2. 配置跨端构建环境
3. 构建对应平台的客户端
